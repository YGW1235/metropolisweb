# 심포지온 홍보 계획

## 서비스 컨셉

심포지온은 일상의 찬반 주제를 가볍게 고르고, 내 선택이 주류인지 비주류인지 확인하며, 짧은 의견으로 편을 드는 캐주얼 논쟁 공간입니다.

## 핵심 메시지

오늘의 주제, 당신은 어느 쪽인가요?

## 타깃 사용자

- 가볍게 투표하고 다른 사람의 생각을 보고 싶은 사용자
- 긴 토론보다 짧은 의견 교환을 선호하는 사용자
- 취향, 일상, 사회 이슈를 부담 없이 이야기하고 싶은 사용자
- 내 선택이 다수인지 소수인지 궁금한 사용자

## SNS 홍보 문구

심포지온에서 가볍게 투표하고 짧은 의견을 남겨보세요. 내 선택이 주류인지 비주류인지도 확인할 수 있습니다.

## 공유 카드 문구

- 오늘의 주제, 당신은 어느 쪽인가요?
- 가볍게 고르고, 짧게 편드는 캐주얼 논쟁 공간
- 내 선택은 주류일까, 비주류일까?

## 대표 주제 준비 기준

- 한눈에 A/B 선택이 가능한 주제
- 너무 무겁지 않지만 의견이 갈릴 만한 주제
- 제목만으로 맥락을 이해할 수 있는 주제
- 댓글과 짧은 의견이 자연스럽게 이어질 수 있는 주제
- 공유했을 때 친구에게 물어보고 싶은 주제

## 홍보 전 체크리스트

- 도메인과 `NEXT_PUBLIC_SITE_URL` 일치 확인
- sitemap, robots, OG 이미지 접근 확인
- 대표 주제 5개 이상 준비
- 문의, 신고, 관리자 moderation 흐름 확인
- 모바일 화면에서 홈, 주제 목록, 주제 상세 확인

## 홍보 후 피드백 수집 방식

- `/contact` 문의로 버그와 제안을 수집합니다.
- 신고 목록에서 운영 이슈를 확인합니다.
- 인기 의견과 댓글 흐름을 보고 주제 품질을 점검합니다.
- Search Console 색인 상태와 유입 검색어를 확인합니다.
